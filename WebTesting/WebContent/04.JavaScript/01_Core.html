1. HTML: 웹페이지를 제작하기 위한 도구이다. 
		2. CSS: HTML을 표현한다. 
		3. JavaScript : HTML 통제하고 유용하게 활용한다. 
			3 - 1) 자바스크립트 구성
					1) CORE : 기본문법구조, 데이터타입, 조건문, 반복문, 함수, 기본라이브러리 등 포함. 
					2) BOM (Browser Object Model) : 브라우저와 관련된 객체 (Window, Location, History, Screen)
					3) DOM (Document Object Model) : Element(요소 태그), Style, Attribute(속성 값) Event 등을 제어한다.
